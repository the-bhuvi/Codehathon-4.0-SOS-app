# Contributing to SafeAlert

First off, thank you for considering contributing to SafeAlert! It's people like you that make SafeAlert such a great tool for the community.

## How Can I Contribute?

### Reporting Bugs
- Reference the issue in your pull request.
- Provide a clear and descriptive title.
- Describe the exact steps which reproduce the problem.

### Suggesting Enhancements
- Use a clear and descriptive title.
- Provide a step-by-step description of the suggested enhancement.
- Explain why this enhancement would be useful.

### Pull Requests
1. Fork the repo and create your branch from `main`.
2. If you've added code that should be tested, add tests.
3. Ensure the test suite passes.
4. Make sure your code lints.
5. Issue that pull request!

## Code of Conduct
Please be respectful and helpful to others in the community.
